# ASNP NATPRO10 in JEJU island

GitHub Pages에서 바로 실행 가능한 Jeju-themed web game.

## 폴더 구조

- `index.html`
- `admin.html`
- `css/style.css`
- `js/main.js`
- `js/game.js`
- `js/audio.js`

## GitHub Pages 배포

1. 이 파일들을 저장소 루트에 업로드합니다.
2. `Settings > Pages`로 이동합니다.
3. `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, Folder는 `/ (root)`를 선택합니다.
5. 저장 후 배포 주소로 접속합니다.

## 중요한 점

기존에 업로드된 파일 중 일부가 **소스 파일이 아니라 GitHub 웹페이지 HTML**로 저장되어 있으면 게임이 실행되지 않습니다.
반드시 실제 `index.html`, `style.css`, `main.js`, `game.js`, `audio.js` 파일을 올려야 합니다.
