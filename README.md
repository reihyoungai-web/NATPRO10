# 🎮 ASNP NATPRO10 in JEJU Island

제주도를 배경으로 한 8비트 레트로 스타일 미니 게임입니다.

## 📋 프로젝트 개요

- **게임명**: ASNP NATPRO10 in JEJU Island
- **장르**: 캐주얼 아케이드 게임
- **플랫폼**: 웹 (PC, 모바일, 태블릿 모두 지원)
- **특징**: 8비트 레트로풍 그래픽, 제주도 테마

## 🎯 게임 규칙

1. **돌하르방**을 터치/클릭하여 위로 던집니다
2. 날아다니는 **한라봉**을 수집하면 +10점
3. **문어**나 **거북이**에 닿으면 게임 오버!
4. 점점 빨라지는 난이도에서 최고 점수에 도전하세요

# 📖 사용 매뉴얼 (초보자용)

## 게임 배포하기

1. **Publish 탭으로 이동**
   - 화면 상단의 "Publish" 탭을 클릭하세요

2. **배포하기**
   - "Publish" 버튼을 클릭하면 자동으로 배포됩니다
   - 배포가 완료되면 URL이 제공됩니다

3. **URL 공유**
   - 제공된 URL을 복사하여 공유하세요
   - 모바일에서도 해당 URL로 접속하면 게임을 플레이할 수 있습니다

## 데이터 다운로드 방법

1. **관리자 페이지 접속**
   - 배포된 URL 뒤에 `/admin.html`을 추가하여 접속
   - 예: `https://your-site.com/admin.html`

2. **CSV 다운로드**
   - "CSV 다운로드" 버튼 클릭
   - 파일이 자동으로 다운로드됩니다

3. **Excel에서 열기**
   - 다운로드된 `.csv` 파일을 Excel에서 열기
   - 이메일과 점수 데이터가 표시됩니다
